## How to automate and streamline document filling

*This chapter provides a step-by-step guide on uploading documents, creating agents, and filling in information on the Platus platform.*

### Step-by-step Guide

To fill a document on Platus, you can use the platform’s features to automate and streamline the process. Here’s how you can do that:

1. Upload your document: If you have a specific document you want to fill out, start by uploading it to the platform.

2. Create an agent: You can create an agent that can help fill in the necessary data points in your document. To do this, follow these steps:
- Go to the agent creation area by clicking on the "Build your Agent" button.
- Use the canvas editor to add, drag, edit, or remove nodes to customize how your agent will operate.

3. Fill in the information: Once your agent is set up to fill out the document, you can provide the required data, and the agent will populate the document accordingly.

To get started right away, you can directly create an agent by clicking here: [Create Agent] (/workflows/editor/new).

If you need any assistance with a specific document or filling it out, feel free to ask!
